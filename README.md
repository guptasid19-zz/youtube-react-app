# youtube-react-app
A simple youtube SPA based on React

INSTALLATION

Run 'npm install' to install all dependencies.

RUNNING THE APPLICATION

Run 'npm start' command to start the app server. Open localhost:8080 to see the app on any web browser.

CONFIGURATION

You will need to add a config.js file in the parent directory folder.

Add this line in config.js file:

export const YOUTUBE_API_KEY = YOUR_YOUTUBE_API_V3_KEY ;
